# 🎨 Theme Generator Pro

Interactive **Theme Generator** built with vanilla JavaScript.
Create custom UI themes with color pickers, sliders and live preview, then export the theme as **CSS variables**.

---

## 🚀 Live Demo

👉 https://johnyisbackk.github.io/js-theme-generator-pro/

---

## ✨ Features

* 🎨 Generate custom UI themes
* 🎚️ Adjust **border radius** and **shadow intensity**
* 🎲 Random theme generator
* 💾 Save themes using **localStorage**
* 📋 Copy generated **CSS variables**
* 👁️ Live preview of UI components
* 🗑️ Delete or clear saved themes

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3**
* **JavaScript (Vanilla JS)**
* **CSS Variables**
* **LocalStorage API**
* **Clipboard API**

---

## 📦 How to Run Locally

Clone the repository and open `index.html`.

```bash
git clone https://github.com/YOUR_USERNAME/theme-generator-pro.git
```

Then open:

```
index.html
```

in your browser.
